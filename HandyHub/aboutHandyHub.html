How HandyHub works (in brief)

To rent Search for the ad you are interested in and check that it is available in the calendar - Send a request and thereby create an account if you do not already have an account (the request is not binding). The landlord approves and you receive a notice if you use the app and an SMS if you use the web. You don't need to pay. You agree on a time for collection and return yourself. You pick up the item at the agreed place and time. You have the item during the booked period. You return the object at the agreed time. To borrow: Post one or more ads. When you receive a request, you accept (or decline if you can't) , The gadget is booked. You agree on a time for extradition and return yourself. You deliver the rental item at the agreed time and place and mark the rental item as collected in HandyHub. The renter returns the rental item and you mark it as returned, which ends the rental and the money is paid to you.
